Compact Tab Bar (30px)

Shorter than current 44px global bar. 11px font. Tabs are pill-shaped, minimal chrome. Active tab has subtle background.

Always-Visible Controls

Split and close buttons permanently in the tab bar's right side. No hover gating. Solves the native-view-over-DOM problem.

Focus = Accent Border

Only the focused split shows an accent-colored bottom border on its tab bar. Click any split to focus it. No hover effects on other splits.

Close Split Button

× button only appears when there are 2+ splits. Closing a split closes all its tabs. Single-pane workspaces don't show ×.

Data model change: Currently Workspace → Tabs → SplitTree → Panes. New model: Workspace → SplitTree → each leaf is a PaneGroup with its own tabs. The "Tab" concept moves from workspace-level to split-group-level.
